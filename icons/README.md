# Icons Directory

This directory should contain the following icon files for the Chrome extension:

- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

These icons will be displayed in the Chrome toolbar and extension management page.

For now, you can use placeholder images or create your own icons with a phishing protection theme (shield, lock, warning symbols, etc.).
